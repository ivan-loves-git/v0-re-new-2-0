## Complete Design System Documentation

This file contains the complete design system for the Renew Dashboard platform, including:

1. **Color Palette** - Primary, neutral, status, and semantic colors
2. **Typography** - Font families, sizes, weights, and hierarchy
3. **Spacing System** - 4px base unit system with tokens
4. **Component Styles** - Buttons, inputs, cards, badges, etc.
5. **Layout Grid** - 12-column responsive grid system
6. **Shadows & Elevation** - 5 levels of elevation
7. **Border Radius** - Consistent rounding standards
8. **Icons** - Heroicons system with size standards
9. **Animations & Transitions** - Timing and easing functions
10. **Accessibility** - WCAG AA compliance standards
11. **Responsive Behavior** - Mobile-first approach
12. **Design Tokens** - CSS variable system
13. **Component Patterns** - Deal cards, KPI widgets, forms
14. **Design Principles** - Asana-inspired core principles

## Quick Reference

### Primary Colors
- Brand Blue: `#2D65F8`
- Brand Teal: `#14B8A6`
- Accent Orange: `#F97316`

### Typography
- Font: Inter
- Base size: 14px
- Scale: 12px → 14px → 16px → 18px → 20px → 24px → 30px → 36px → 48px

### Spacing
- Base unit: 4px
- Common: 8px, 12px, 16px, 24px, 32px, 48px

### Components
- Border radius: 6px (default), 8px (cards), 12px (modals)
- Button padding: 10px × 20px
- Card padding: 20px
- Input padding: 10px × 12px

###Shadows
- Subtle: `0 1px 3px rgba(0, 0, 0, 0.05)`
- Medium: `0 4px 12px rgba(0, 0, 0, 0.08)`
- Raised: `0 8px 24px rgba(0, 0, 0, 0.12)`
